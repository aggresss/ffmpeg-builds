# ffmpeg-builds

## Build

```
git clone https://github.com/aggresss/ffmpeg-builds.git
cd ffmpeg-builds
cmake . -Bbuild && make -C build
```

## Diagnostics

`./build/unix/ffmpeg-prefix/src/ffmpeg/ffbuild/config.log`
